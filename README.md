# SLA Breach Prediction using Machine Learning

## Problem
Predict whether a support ticket will breach SLA.

## Approach
- Performed EDA to understand patterns
- Created features like Load and Peak Hour
- Built Logistic Regression model
- Evaluated using accuracy and confusion matrix

## Tools Used
Python, Pandas, Scikit-learn

## Outcome
Model predicts SLA breach risk based on input features.

- Add more real-world features
- Improve accuracy using advanced models
